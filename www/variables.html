Summary

The term floating point is derived from the fact that there is no fixed number of digits before and after the decimal point; that is, the decimal point can float.

Which of the following is not a floating point number?

A. 210.5628
B. 12,457
C. 1.6x10e-7
D. 0.0971

That's Correct!

B. 12,457

Clearly 12,457 is an integer and not a floating point number. Always look for a decimal while picking out floating point numbers
